# Beaver-Tools
Odoo Support tools to help condense logfiles to a single database


# First Time Configuration
Everything should run on download.  To correct the image for the app icon, change the path from =/home/odoo/Beaver-Tools/beaver.jpg to the location of your file
# To Run
Double click on TheBeaver.desktop   Then, type the name of the database that you want, and enjoy your filtered file
