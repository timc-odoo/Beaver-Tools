# Beaver-Tools
Odoo Support tools to help condense logfiles to a single database


# First Time Configuration
From the beaver.SH file you will need to edit line 10 to the location that you store the .PY file if you do not keep it in the same location
From the beaver.PY file you will need to edit line 9 to where you would like the output of the file

# To Run
Either from the location of the beaver.SH file, or via an alias created in your .bashrc file call the .SH file and select the file you want to filter.  Then, type the name of the database that you want, and enjoy your filtered file
